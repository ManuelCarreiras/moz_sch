## Best Practices

1. **Set up criteria BEFORE grading** - Students can't be graded without criteria
2. **Weights must total 100%** - System validates this
3. **Be consistent** - Use same component names (Tests, Homework, Attendance)
4. **Different years can have different rules** - 1st year vs 2nd year Algebra
